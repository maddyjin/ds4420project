Legislative Voting Behavior Prediction
Madeleine Jin + Arielle Rabinovich
DS 4420 Final Project

This project aims to predict how individual legislators will vote on bills using political, social, and historical signals. We plan to analyze various data points in order to maximize the accuracy of our model, and doing so by using past voting records and contextual information. We will develop and test various machine learning models that estimate the likelihood of a legislator voting Yes, No, or Abstain on new legislation, and analyze which model is best suited to answer our question.

Broadly, the overarching goal of our project is to deepen our understanding of legislative voting behaviors and how it is related to political alignment, party influence, and policy trends.
